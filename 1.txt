Shell scripting:

For loop:
    syntax:
    for ((initialization;condition;inc/dec))
    do
    statements
    done

If statement:
    syntax:
        if[condition]
        then
        command
        else
        command
        fi

Wile loop:
    syntax:
        initialization;
        while(condition)
        do
        block of code
        action
        done

switch case:
    syntax:
        case expression in
        pattern 1)
        statements;;
        pattern 2)
        statements;;
        .
        .
        .
        *) (default)
        esac

tail:
